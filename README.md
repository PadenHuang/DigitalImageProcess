│  │  │          └─view  视图

└─target  编译文件目录

#### 六、使用

1、git clone到本地之后可以直接在IDEA上打开，注意配置开发环境。

2、使用“数字图像处理小工具.exe”文件——双击该.exe文件即可使用。

3、使用jar包文件——cmd 命令行运行“java -jar DigitalImageProcessLearn.jar”即可使用（需要注意的是cmd运行时必须要进入到jar包指定的目录才可使用)。
